Questions

FAQ

How long does a typical project take? +

Micro-interaction packages: 1-2 weeks. Prototype motion projects: 2-4 weeks. Enterprise systems: 6-10 weeks with iterative reviews.

Do you work on retainer? +

Yes, for select enterprise clients requiring ongoing motion system maintenance, icon updates, or new screen animations each sprint.

What files do I receive? +

Depending on package: Figma source files, Protopie / Framer prototypes, Lottie JSON files, CSS custom properties, React Framer Motion code snippets, and a full PDF motion spec.

Do you require a deposit? +

Yes — 50% upfront to schedule and begin work, 50% upon delivery and final approval.
